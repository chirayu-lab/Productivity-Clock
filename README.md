## How to Use

### Timer
1. Click **Focus** or **Break** to select your mode
2. Press the **Play** button to start the timer
3. The circular progress ring shows your progress
4. Press **Pause** to pause, or the reset button (circle icon) to start over
5. Timer automatically switches modes when complete

### Tasks
1. View your current tasks in the Quick Tasks section
2. Click the checkbox to mark tasks complete
3. Hover over tasks to reveal the delete button
4. Add new tasks using the input field at the bottom
5. Press Enter to quickly add a task
